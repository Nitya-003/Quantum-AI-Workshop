# Quantum Circuit using PennyLane 

This project demonstrates a simple quantum circuit built using PennyLane.

## Features
- RZ and RY rotation gates
- CNOT entanglement
- Expectation value measurement
- Circuit visualization

## Output
The circuit returns expectation value of Pauli-Z on qubit 1.
